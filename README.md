# Mode Manager (Subway Builder Mod)

An all-in-one Subway Builder mod that bundles all custom transit modes. Players choose which modes are active from an in-game UI panel — no manual mod switching required.

## Included Modes

| Mode | Description |
|---|---|
| Tram | Lightweight tram for short-distance urban transit |
| Bus Rapid Transit | High-capacity bus for dedicated rapid transit corridors |
| Monorail | Elevated monorail for scenic urban and resort transit |
| People Mover | Automated people mover for short-distance elevated transit |

## Installation

**Mod files are located in `mode-manager/` in this repository.** Follow these steps to install:

### Install on Your System

The mod installs to your Subway Builder mods folder. Platform-specific locations:

| Platform | Mod Folder Path |
|----------|-----------------|
| **Windows** | `%APPDATA%/metro-maker4/mods/` |
| **macOS** | `~/Library/Application Support/metro-maker4/mods/` |
| **Linux** | `~/.config/metro-maker4/mods/` |

**Quick method:**
1. In Subway Builder, open Settings > System > Open Saves Folder
2. Copy the `mode-manager` folder from this repository into your mods folder

**Manual method:** See [Installation Guide](docs/mod-installation.md) for detailed platform-specific instructions.

### Enable the Mod

1. Launch Subway Builder
2. Open Settings > Mods
3. Find **Mode Manager** in the mods list
4. Toggle the mod to **enabled**
5. Restart the game
