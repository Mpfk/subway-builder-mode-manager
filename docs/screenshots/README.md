# Screenshots

Drop the README image files here. The main README references these paths:

| File | Used in | Suggested width |
|---|---|---|
| `this-save.png` | README hero (under the tagline) | Full-width landscape |
| `library.png` | README → Using Mode Manager → Library tab | ~450 px (portrait) |
| `editor.png` | README → Using Mode Manager → Mode editor | ~450 px (portrait) |

Keep PNG dimensions reasonable — GitHub renders the README at a fixed container width, so anything past ~2000 px wide just wastes bandwidth. Crop to the panel chrome and a bit of game canvas around it where applicable.
